# Bank Account Management System

## Project Description
The Bank Account Management System is a Java-based console application developed to perform basic banking operations. It allows users to create an account, deposit money, withdraw money, and check their account balance.

## Features
- Account Creation
- Deposit Amount
- Withdraw Amount
- Balance Checking
- Input Validation
- Error Handling
- Menu-Driven Interface

## Technologies Used
- Java
- VS Code
- Scanner Class

## How to Run
1. Compile the program:
   javac BankAccountManagementSystem.java

2. Run the program:
   java BankAccountManagementSystem

## Sample Operations
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Exit Application
