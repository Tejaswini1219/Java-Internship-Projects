# Java Internship Projects

## Overview

This repository contains Java projects completed as part of the IncodeVision Java Internship Program. These projects were developed to strengthen core Java programming skills, problem-solving abilities, Object-Oriented Programming concepts, and practical application development.

The repository includes the following projects:

1. Bank Account Management System
2. Simple Quiz Application

---

# Task 01 - Bank Account Management System

## Project Description

The Bank Account Management System is a console-based Java application that simulates basic banking operations. The system allows users to create an account, deposit money, withdraw money, and check their account balance through a menu-driven interface.

This project demonstrates how real-world banking operations can be implemented using Java programming concepts.

## Features

- Account Creation
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Input Validation
- Error Handling
- Menu-Driven Interface
- User-Friendly Console Interaction

## Technologies Used

- Java
- VS Code
- Scanner Class
- Conditional Statements
- Loops
- Methods

## Learning Outcomes

Through this project, I learned:

- Java Fundamentals
- Control Structures
- Methods and Functions
- User Input Handling
- Validation Techniques
- Real-World Problem Solving

# Task 02 - Simple Quiz Application

## Project Description

The Simple Quiz Application is a Java-based console application that presents multiple-choice questions to users, collects responses, evaluates answers, and calculates the final score.

Questions and answers are stored using Java Collections, helping users understand collection frameworks and Object-Oriented Programming concepts.

## Features

- Multiple Choice Questions (MCQs)
- Interactive User Interface
- Score Calculation
- Final Result Display
- Question Management using ArrayList
- Object-Oriented Programming Concepts
- User Response Evaluation

## Technologies Used

- Java
- ArrayList Collection
- OOP Concepts
- Scanner Class
- Loops and Conditional Statements

## Learning Outcomes

Through this project, I learned:

- Java Collections Framework
- ArrayList Implementation
- Object-Oriented Programming
- User Interaction Design
- Data Handling
- Logic Building and Problem Solving
